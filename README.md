## How to use

A little shoutout to Aunt Bethany of Christmas Vacation... "Come on, Aunt Bethany, let's say hello everybody."

```python
say_hello('everybody')
```




    'Hello everybody!'



There are a couple of Easter eggs in this function.

```python
say_hello('Jimmy McGill')
```




    "S'all good, man."



There's a class, too!

```python
greet = HelloSayer('everybody')
greet.say()
```




    'Hello everybody!'



```python
HelloSayer('Jesse Pinkman').say()
```




    'Yo'
